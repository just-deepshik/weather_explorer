# Weather Explorer 

Weather Explorer is a Python Tinkter desktop application that provides real-time weather information for any city using OpenWeathermap API

## Features
- Real-time weather data
- Temperature, humidity, wind speed
- Sunrise time
- User-friendly Tkinter GUI

## Technologies Used
- Python
- Tkinter
- OpenWeather API
- Requests library

## Installation

pip install requests

## API Key Setup

This project uses the OpenWeather API.

1. Create a free account at https://openweathermap.org/
2. Generate your API key.
3. Open the file `weather_explorer.py`
4. Replace the API key in the following line:

API_KEY = "YOUR_API_KEY_HERE"

Example:

API_KEY = "abcd1234yourapikey"

## Run

python weather_explorer.py
